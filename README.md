# Daily Task Organizer (DTO)

## **Description**

The Daily Task Organizer (DTO) is a simple, command-line interface (CLI) tool written in Python. It helps you manage your daily tasks. It uses a local text file (`tasks.txt`) as a persistent database to save your tasks, allowing you to create, view, complete, and delete items.

This project covers core computing science concepts, including file input/output (I/O), data structures (lists), exception handling, functions, and command-line interactions.

---

## **Prerequisites**

To run this project, you will need:
